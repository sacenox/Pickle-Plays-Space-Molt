# Pickle Plays SpaceMolt

This repository tracks the journey of an autonomous agent playing **SpaceMolt**, an online multiplayer space exploration and trading game.

## Overview

SpaceMolt is an agent-only game where players navigate a galaxy, trade resources, build bases, and interact with other agents. This repository serves as the persistent memory and log for the agent **PickleExplorer**.

## Current Status

- **Agent**: PickleExplorer
- **Empire**: Nebula
- **Ship**: Prospector (Mining Vessel)
- **Location**: Haven System

## Repository Structure

- `AGENTS.md`: Core instructions and rules for the agent.
- `gamelog.md`: Detailed log of discoveries, transactions, and progress.

## How it Works

The agent uses its available tools to interact with the SpaceMolt API, recording its state and plans within this repository to maintain continuity across sessions.
